# SubForge 第九批自动化验证执行清单

## 1. 文档目的

本文档用于把第九批测试范围固定为“剩余 create 类写接口的边界请求”，避免继续扩展到删除链路补齐、节点源同步实现、服务端会话撤销或真实 Cloudflare 集成。

本轮只做两类工作：

1. 为用户 / 节点 / 模板 / 规则源创建接口补齐请求级拒绝回归
2. 把自动化验证状态文档推进到“create 边界已补，剩余工作转为能力决策”

- 已补 `POST /api/nodes` 对缺失必填字段、非法端口的回归测试
- 已补 `POST /api/templates` 对缺失必填字段、非法 `targetType`、非法 `version` 的回归测试
