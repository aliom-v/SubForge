- `apps/web/src/node-metadata.ts`

- `credentials.password`
- `params.sni`
- `params.obfs`
- `params["obfs-password"]`

- 复杂变体继续写实标注，不宣称“全协议完成”

### 3.2 `ss` / `hysteria2` 协议级字段校验

已落地到：

- `packages/core/src/node-protocol-validation.ts`
- `packages/core/src/node-import.ts`
- `apps/web/src/App.tsx`
- `apps/worker/src/index.ts`

当前语义：

- `ss` 至少要求 `credentials.cipher` 与 `credentials.password`
- `ss` 对复杂 plugin 字段给出稳定 unsupported 提示
- `hysteria2` 至少要求 `credentials.password`
- `hysteria2` 对未知 query 参数、非法 `obfs`、错误类型字段给出显式报错
- `hy2` 会统一规范化为 `hysteria2`
