# SubForge 第十批自动化验证执行清单

## 1. 文档目的

本文档用于把第十批测试范围固定为“删除链路补齐与最小回归保护”，避免继续扩展到节点源同步、服务端会话撤销或真实 Cloudflare 集成。

本轮只做三类工作：

1. 补齐用户 / 模板 / 规则源删除接口
2. 为删除成功与缺失资源补请求级回归
3. 同步 README、部署与路线文档中的真实能力描述

当前进展：`done`

- 已补 `DELETE /api/users/:id`，成功后失效用户缓存、写审计、旧 token 访问返回 `404`
- 已补 `DELETE /api/templates/:id`，仅在生效模板变化时按目标类型失效缓存
- 已补 `DELETE /api/rule-sources/:id`，成功后失效全部用户缓存并删除对应快照
- 已补对应 `404` 与缓存 / 审计回归测试
- 已补前端删除操作按钮与 API 封装
