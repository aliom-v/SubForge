# SubForge 第十三批自动化验证执行清单

## 1. 文档目的

本文档用于把第十三批测试范围固定为“禁用中的默认模板不会继续保留默认标记”的语义收口，避免继续扩展到模板版本策略、节点源同步或更大的订阅编译改造。

本轮只做三类工作：

1. 收紧模板更新持久化不变量，避免 disabled 模板残留 `is_default=1`
2. 补 `PATCH /api/templates/:id` 禁用当前默认模板时的请求级回归
3. 同步自动化验证、状态与路线文档，继续保持模板返回状态与实际生效选择一致

当前进展：`done`

- 已补模板更新持久化不变量：默认模板在被禁用时会自动清除默认标记
- 已补 `PATCH /api/templates/:id` 禁用当前默认模板的请求级回归
- 已确认响应体、DB 持久化状态、缓存失效与审计写入保持一致
